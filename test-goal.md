# /goal 功能测试指南

## 测试步骤

1. 启动应用：
   ```bash
   npm run start
   ```

2. 在输入框中输入以下命令：
   ```
   /goal 创建一个简单的计算器应用
   ```

3. 观察执行过程：
   - 系统会显示"启动目标执行系统..."
   - 系统会分析目标并制定执行计划
   - 显示子任务列表和预计时间
   - 多个代理并行执行子任务
   - 实时显示进度和状态

## 预期结果

1. **目标规划阶段**：
   - 显示"目标规划中 - 正在分析目标并制定执行计划..."
   - 显示"执行计划已创建"，包含子任务数量和预计时间

2. **任务执行阶段**：
   - 显示"子任务开始执行"
   - 显示"子任务进行中 - 进度: XX%"
   - 显示"子任务完成"

3. **目标完成阶段**：
   - 显示"目标完成!"
   - 显示执行结果总结

## 故障排除

如果遇到问题，请检查：

1. **API 配置**：
   - 确保 API Key 正确
   - 确保 Base URL 正确
   - 确保模型名称正确

2. **网络连接**：
   - 确保可以访问 API 服务器
   - 确保没有防火墙阻止

3. **控制台日志**：
   - 打开开发者工具查看控制台日志
   - 查看是否有错误信息

## 功能特点

- **智能规划**：自动分析目标并制定执行计划
- **并行执行**：多个代理同时执行独立子任务
- **依赖管理**：自动处理子任务之间的依赖关系
- **实时反馈**：实时显示执行进度和状态
- **错误处理**：自动重试失败的子任务
